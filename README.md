# Motorola Sales Dashboard

📊 Motorola Sales Dashboard

## Overview

This Power BI dashboard provides a comprehensive view of Motorola’s sales performance across India. It visualizes key metrics such as total sales, quantity sold, transactions, average sales, and customer ratings — helping stakeholders analyze trends, regional performance, and product insights effectively.

## Features

- Interactive maps and region-wise performance visualizations
- Time-series trends for sales, transactions, and quantities
- Product-level breakdowns and top-selling SKUs
- Customer rating summaries and distribution
- Filters for state, city, product family, and date ranges
- Exportable reports and visual snapshots

## Data

The dashboard expects a dataset containing sales transactions with fields such as:

- Transaction ID
- Date
- State
- City
- Product Name / SKU
- Quantity
- Sales Amount
- Customer Rating

If you used a sample dataset or CSV, include it in the repository (e.g., data/sales.csv) and update the data source path in Power BI.

## How to use

1. Open the Power BI Desktop file (.pbix) included in this repository (if present).
2. If the data source is local (CSV/Excel), update the data source path to point to your file.
3. Refresh the dataset to load current data.
4. Use the slicers and filters on the report to explore regional or product-level insights.
